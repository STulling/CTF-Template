# Explanation

## Use this template to create your own CTF challenges. 
You can use the `example-offline` and `example-online` folders as a starting point for your own challenges. Just copy the folder, rename it, and edit the `challenge.yml` file and any other files you need to create your challenge.

The `example-online` is a simple web challenge that can be deployed using Docker, this one will have a hosted instance attached to it for people trying to solve it.

The `example-offline` is a simple offline challenge that includes a text file, this one will not have a hosted instance attached to it, and is meant to be downloaded and solved locally by the participants.
